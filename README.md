# Hermès Claw

> A game about wanting things.

You want a Birkin bag. She decides who gets one.

## About

Hermès Claw is a browser-based card game set inside a Hermès boutique. Each turn you choose one of three cards — buy something, chat up the associate, or walk away. Your choices build (or erode) goodwill over 10 turns. If you've played it right, she offers you the bag. If you haven't, you discover what happens instead.

There is a pattern. You'll feel it before you understand it.

## Built With

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
